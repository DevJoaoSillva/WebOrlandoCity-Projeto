# Site sobre Orlando City

Este é um site completo dedicado à cidade de Orlando, Flórida, nos Estados Unidos. O site abrange uma variedade de informações sobre a cidade, incluindo atrações turísticas, eventos locais, dicas de viagem e muito mais.

## Conteúdo

O site inclui as seguintes seções principais:

- Página inicial: Introdução ao Orlando City Soccer Club e destaques principais.
- História: Uma visão geral da história do clube, desde sua fundação até os dias atuais.
- Notícias: Últimas notícias e atualizações relacionadas ao Orlando City Soccer Club.


## Tecnologias Utilizadas

O site foi desenvolvido utilizando as seguintes tecnologias:

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Slim Framework 2

## Como Utilizar

Para visualizar o site localmente, siga estas etapas:

1. Clone este repositório em sua máquina local.
2. Configure um servidor web local (como Apache) com suporte a PHP.
3. Coloque todos os arquivos do projeto no diretório raiz do seu servidor web.
4. Abra o navegador da web e acesse `http://localhost/index.php`.

Alternativamente, você pode hospedar o site em um servidor web para torná-lo acessível online. Para isso, siga estas etapas:

1. Adquira um serviço de hospedagem web com suporte a PHP.
2. Faça o upload dos arquivos do site para o servidor web, utilizando um cliente FTP ou ferramenta semelhante.
3. Acesse o domínio ou URL associado ao seu serviço de hospedagem para visualizar o site online.
